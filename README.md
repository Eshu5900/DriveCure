# 🚗 DriveCure - Intelligent Vehicle Problem Detection System

DriveCure is a full-stack AI-powered web application that helps users detect and diagnose vehicle problems instantly using intelligent analysis.

## 🔥 Features
- AI-based vehicle issue detection  
- Exact root cause identification  
- Repair & replacement cost estimation (INR)  
- DIY steps guidance  
- Voice input support 🎤  
- Nearby mechanic finder 📍  

## 🛠 Tech Stack
- Frontend: React  
- Backend: Node.js + Express  
- Database: MongoDB  
- AI: OpenAI API  
- Deployment: Render (Backend) + Netlify (Frontend)  

## 🌐 Live Demo
https://magical-rugelach-001c7d.netlify.app

## 📂 Project Structure
- frontend/ → User interface (React)  
- backend/ → Server, API & database  

## 🚀 How It Works
1. User enters vehicle problem (text or voice)  
2. Backend processes input using OpenAI  
3. AI returns:
   - Problem summary  
   - Root cause  
   - Solutions  
   - Cost estimation  
4. Data is stored in MongoDB for history
